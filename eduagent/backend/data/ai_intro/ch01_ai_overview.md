# 人工智能概述

## 1. 什么是人工智能

人工智能（AI）是让机器模拟人类智能的学科。从简单的规则系统到当前的深度学习和语言模型，AI经历了70多年的发展。

## 2. AI的分类

### 按能力分类
- **弱人工智能**：专注于特定任务（如图像识别、语音助手）
- **强人工智能**：具有通用智能，能完成任何人类能做的认知任务（尚未实现）
- **超人工智能**：超越人类智能（理论阶段）

### 按方法分类
- **符号主义**：基于逻辑推理和知识表示
- **连接主义**：基于神经网络和深度学习
- **行为主义**：基于感知和行动的学习

## 3. AI的发展历程

| 阶段 | 时期 | 代表 |
|------|------|------|
| 诞生 | 1956 | 达特茅斯会议 |
| 第一次寒冬 | 1970s | 计算力不足 |
| 专家系统 | 1980s | 基于规则 |
| 第二次寒冬 | 1990s | 维护困难 |
| 深度学习 | 2012- | ImageNet突破 |
| 大模型时代 | 2020- | GPT、BERT |

## 4. AI的主要应用

- 计算机视觉：图像识别、目标检测
- 自然语言处理：翻译、对话、文本生成
- 语音识别：语音助手、转录
- 推荐系统：电商、视频推荐
- 自动驾驶：感知、决策、控制

## 5. 关键要点
1. AI不是单一技术，是多学科的交叉
2. 深度学习的成功依赖于大数据和强算力
3. 当前AI仍处于弱人工智能阶段
4. AI的核心是"学习"，而非"编程"
