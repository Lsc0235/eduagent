智学通 EduAgent 打开说明

最推荐的打开方式：

1. 先把压缩包完整解压出来，不要直接在压缩包里面双击文件。
2. 进入解压后的 dasai 文件夹。
3. 第一次运行可以直接双击“启动智学通.bat”。
   - 如果项目本地环境不存在，启动脚本会自动调用“首次安装依赖.bat”。
   - 也可以手动先双击“首次安装依赖.bat”，等它安装完成后再启动。
4. 浏览器会自动打开：
   http://127.0.0.1:5173/learning

电脑需要提前具备：

- Node.js LTS 版本。
- Python 3.10/3.11，或 Anaconda / Miniconda。

项目会优先使用 dasai 文件夹里的本地环境：

- .venv
- 或 .conda-env

因此不要把自己电脑上的 Python 绝对路径写进代码，也不要手动改启动路径。

互相打包给队友时：

1. 双击“打包给队友.bat”。
2. 它会生成 dasai-portable-日期时间.zip。
3. 把这个 zip 发给队友即可。

不要手动把这些本机文件放进压缩包：

- .venv
- .conda-env
- node_modules
- dist
- .git
- .env
- *.db
- 各种 __pycache__、日志、缓存文件

注意：

- 启动后会弹出 Backend 和 Frontend 两个黑色窗口，不要关闭。
- 如果页面打不开，等 10 秒再刷新浏览器。
- 如果提示端口被占用，先关闭之前打开的 Backend / Frontend 黑窗口，再重新双击启动。
- 如果安装依赖失败，通常是 Python / Node.js 没装好，或者网络无法下载依赖。
