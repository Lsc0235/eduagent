# 模型评估与交叉验证

## 1. 概述

模型评估是选择最佳模型和参数的关键步骤。交叉验证是评估泛化能力的重要手段。

## 2. 评估指标

### 分类指标
- **准确率**：正确预测数÷总预测数
- **精确率**：TP/(TP+FP)，预测为正的样本中真正正的比例
- **召回率**：TP/(TP+FN)，实际为正的样本中被找出的比例
- **F1分数**：精确率和召回率的调和平均

### 回归指标
- **MSE**：均方误差
- **MAE**：平均绝对误差
- **R²**：决定系数（越接近1越好）

## 3. 交叉验证

### K折交叉验证
1. 将数据分成K份
2. 每份轮流作为验证集
3. 其余作为训练集
4. 取K次评估结果的平均值

### 留出法
按比例划分训练集和测试集（如7:3）。

## 4. 偏差-方差权衡

- **偏差**：模型预测与真实值的差距
- **方差**：模型对不同训练集的敏感度
- 简单模型：高偏差低方差
- 复杂模型：低偏差高方差

## 5. 关键要点
1. 永远不要用测试集调参
2. K=5或10是最常用的折数
3. 不平衡数据要关注F1而非准确率
4. 评估指标的选择取决于业务场景
