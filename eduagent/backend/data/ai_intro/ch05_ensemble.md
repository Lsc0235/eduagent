# 集成学习

## 1. 概述

集成学习通过组合多个弱学习器来构建强学习器。核心思想是"三个臭皮匠，顶个诸葛亮"。

## 2. 主要方法

### Bagging（Bootstrap Aggregating）
- 有放回抽样创建多个子集
- 并行训练多个模型
- 最终结果通过投票（分类）或平均（回归）
- 代表：随机森林

### Boosting
- 串行训练，每个模型纠正前一个的错误
- 给错分样本更高权重
- 代表：AdaBoost、XGBoost、LightGBM

### Stacking
- 用多个基学习器预测
- 用元学习器融合基学习器的输出

## 3. 随机森林 vs XGBoost

| 特征 | 随机森林 | XGBoost |
|------|---------|---------|
| 方法 | Bagging | Boosting |
| 训练方式 | 并行 | 串行 |
| 准确性 | 较高 | 通常更高 |
| 训练速度 | 快 | 慢 |
| 可解释性 | 较好 | 较差 |

## 4. 关键要点
1. 集成学习是工业界的首选方法
2. 随机森林适合快速原型
3. XGBoost/LightGBM 适合追求极致精度
4. 单一模型很少能赢过集成模型
