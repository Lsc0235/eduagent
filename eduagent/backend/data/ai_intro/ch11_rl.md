# 强化学习基础

## 1. 概述

强化学习通过与环境交互，学习最优决策策略。核心是"试错学习"——智能体采取行动，获得奖励，逐步优化。

## 2. 核心概念

- **智能体（Agent）**：学习者和决策者
- **环境（Environment）**：智能体所处的外部世界
- **状态（State）**：环境的当前情况
- **动作（Action）**：智能体的行为
- **奖励（Reward）**：环境对动作的反馈
- **策略（Policy）**：状态到动作的映射

## 3. 马尔可夫决策过程（MDP）

(S, A, P, R, γ)
- S: 状态空间
- A: 动作空间
- P: 状态转移概率
- R: 奖励函数
- γ: 折扣因子

## 4. Q-Learning

核心公式：
Q(s,a) ← Q(s,a) + α[r + γ·max Q(s',a') - Q(s,a)]

- α: 学习率
- γ: 折扣因子
- 通过不断更新Q值表来学习最优策略

## 5. 关键要点
1. 强化学习的核心是试错学习
2. Q-Learning是入门算法
3. 深度强化学习（DQN）结合神经网络
4. 应用：游戏AI、机器人控制、自动驾驶
