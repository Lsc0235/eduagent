# 自然语言处理基础

## 1. 概述

自然语言处理（NLP）是让计算机理解、生成人类语言的技术。

## 2. 文本预处理

- **分词**：将文本切分为词语或子词
- **去除停用词**：去掉"的""了"等无意义词
- **词干提取**：还原词语的基本形式
- **向量化**：将文本转为数值向量

## 3. 词向量

### Word2Vec
- CBOW: 用上下文预测中心词
- Skip-gram: 用中心词预测上下文

### GloVe
基于词共现矩阵的全局方法

## 4. 预训练模型

| 模型 | 特点 | 年份 |
|------|------|------|
| BERT | 双向Transformer | 2018 |
| GPT | 自回归生成 | 2018 |
| T5 | Text-to-Text框架 | 2019 |

## 5. 关键要点
1. NLP从规则方法发展到深度学习
2. BERT/GPT等预训练模型是当前主流
3. 大语言模型（LLM）是NLP的巅峰应用
4. 分词是中文NLP的独特挑战
