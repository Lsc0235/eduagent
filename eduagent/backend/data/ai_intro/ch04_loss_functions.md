# 损失函数

## 1. 概述

损失函数衡量模型预测值与真实值之间的差距，是训练过程中的优化目标。

## 2. 回归损失函数

### MSE（均方误差）
MSE = (1/n)·Σ(yᵢ - ŷᵢ)²
- 对异常值敏感
- 梯度与误差成正比

### MAE（平均绝对误差）
MAE = (1/n)·Σ|yᵢ - ŷᵢ|
- 对异常值更鲁棒
- 梯度恒定

## 3. 分类损失函数

### 交叉熵损失
L = -Σ yᵢ·log(pᵢ)
- 概率越不准确，损失越大
- 是最常用的分类损失

### 铰链损失（Hinge Loss）
L = max(0, 1 - y·ŷ)
- SVM 的标准损失
- 鼓励分类间隔最大化

## 4. 如何选择

| 任务 | 推荐损失函数 |
|------|------------|
| 回归 | MSE 或 MAE |
| 二分类 | 交叉熵（BCE） |
| 多分类 | 分类交叉熵（CCE） |
| SVM | 铰链损失 |

## 5. 关键要点
1. 损失函数的选择直接影响模型性能
2. 交叉熵是分类任务的标准选择
3. 损失函数必须是可导的（用于梯度下降）
