# 梯度下降

## 1. 概述

梯度下降是机器学习中最基础的优化算法，通过迭代地向负梯度方向移动参数，使损失函数逐步减小。

## 2. 基本公式

参数更新: θ = θ - α · ∇J(θ)
- θ: 模型参数
- α: 学习率
- ∇J(θ): 损失函数对参数的梯度

## 3. 三种变体

### 批量梯度下降（BGD）
每次使用全部训练数据计算梯度。收敛稳定但计算量大。

### 随机梯度下降（SGD）
每次随机选一个样本计算梯度。计算快但震荡大。

### 小批量梯度下降（Mini-batch GD）
每次使用一小批样本。兼顾效率和稳定性，最常用。

## 4. 学习率选择

- 太大：可能震荡发散
- 太小：收敛太慢
- 常用策略：学习率衰减（如余弦退火）

## 5. 动量法

引入动量项，加速收敛，减少震荡：
v = β·v + (1-β)·∇J(θ)
θ = θ - α·v

## 6. 关键要点
1. 梯度下降是神经网络的训练基础
2. Mini-batch是最实用的选择
3. 学习率是最重要的超参数
4. Adam优化器结合了动量和自适应学习率
