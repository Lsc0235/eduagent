# K-Means 聚类

## 1. 概述

K-Means是最经典的无监督聚类算法，将数据分成K个簇，使簇内距离最小化。

## 2. 算法步骤

1. 随机选择K个初始聚类中心
2. 将每个样本分配到最近的聚类中心
3. 更新聚类中心为簇内样本均值
4. 重复2-3直到收敛

## 3. 如何选择K值

- **肘部法则**：观察SSE随K变化的曲线
- **轮廓系数**：评估聚类质量
- **业务需求**：根据实际需求确定

## 4. 优缺点

### 优点
- 简单快速，适合大规模数据
- 可解释性强

### 缺点
- 需预设K值
- 对初始中心敏感
- 不适合非球形分布

## 5. 关键要点
1. K-Means是最常用的聚类算法
2. 特征缩放很重要（距离敏感）
3. 肘部法则是选择K值的常用方法
4. 多次运行取最优结果
