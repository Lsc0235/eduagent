# 机器学习基础

## 1. 什么是机器学习

机器学习是人工智能的一个分支，使计算机能够从数据中学习，而无需显式编程。核心思想是：从数据中发现模式，用这些模式对新数据进行预测。

## 2. 学习类型

### 监督学习
- 有标签数据：输入X → 输出Y
- 分类：离散输出（如垃圾邮件/正常邮件）
- 回归：连续输出（如房价预测）

### 无监督学习
- 无标签数据，发现隐藏结构
- 聚类：将数据分组（如客户分群）
- 降维：减少特征数量（如PCA）

### 强化学习
- 通过与环境交互学习最优策略
- 智能体采取行动，获得奖励
- 目标是最大化累积奖励

## 3. 模型评估

### 分类指标
- 准确率(Accuracy)：正确预测数/总预测数
- 精确率(Precision)：TP/(TP+FP)
- 召回率(Recall)：TP/(TP+FN)
- F1分数：精确率和召回率的调和平均

### 回归指标
- MSE：均方误差
- MAE：平均绝对误差
- R²：决定系数

## 4. 过拟合与欠拟合

### 过拟合
模型在训练集上表现好，但泛化能力差。
解决：正则化、增加数据、减少模型复杂度

### 欠拟合
模型在训练集上就表现差。
解决：增加模型复杂度、增加特征、减少正则化

## 5. 关键要点

1. 机器学习的核心是从数据中学习模式
2. 选择合适的评估指标很重要
3. 过拟合是实际应用中最常见的问题
4. 数据质量比模型复杂度更重要
