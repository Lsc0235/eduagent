# 主成分分析（PCA）

## 1. 概述

PCA是最常用的降维算法，通过线性变换将高维数据投影到低维空间，同时最大限度保留原始信息。

## 2. 核心思想

找到数据方差最大的方向（主成分），将数据投影到这些方向上。

## 3. 算法步骤

1. 数据中心化（减去均值）
2. 计算协方差矩阵
3. 计算特征值和特征向量
4. 选择前k个最大特征值对应的特征向量
5. 将数据投影到选定的特征向量上

## 4. 用途

- **降维**：减少特征数量，加快训练速度
- **可视化**：将高维数据投影到2D/3D
- **去噪**：丢弃小特征值对应的噪声分量
- **特征提取**：生成不相关的新特征

## 5. Python示例

```python
from sklearn.decomposition import PCA

pca = PCA(n_components=2)
X_reduced = pca.fit_transform(X)
print(f"保留方差比例: {sum(pca.explained_variance_ratio_):.2%}")
```

## 6. 关键要点
1. PCA是无监督的线性降维方法
2. 使用前记得标准化数据
3. 特征值大小表示该主成分的重要性
4. PCA假设数据是线性结构
