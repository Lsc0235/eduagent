# 深度学习

## 1. 概述

深度学习是机器学习的一个子领域，使用多层神经网络来学习数据的层次化表示。近年来在图像识别、语音识别、自然语言处理等领域取得了突破性进展。

## 2. 卷积神经网络（CNN）

### 2.1 核心组件

- **卷积层（Convolutional Layer）**：通过卷积核提取局部特征
- **池化层（Pooling Layer）**：降低特征图尺寸，提取主要特征
- **全连接层（Fully Connected Layer）**：进行最终分类或回归

### 2.2 卷积操作

输入特征图 * 卷积核 = 输出特征图

关键参数：
- **卷积核大小**：常用 3x3、5x5
- **步长（Stride）**：卷积核移动的步幅
- **填充（Padding）**：在输入边缘补零
- **通道数（Channels）**：RGB 图像通常有 3 个通道

### 2.3 经典架构

| 模型 | 年份 | 特点 |
|------|------|------|
| LeNet-5 | 1998 | 手写数字识别的开创性工作 |
| AlexNet | 2012 | ReLU、Dropout、GPU 训练 |
| VGGNet | 2014 | 小卷积核堆叠（3x3） |
| GoogLeNet | 2014 | Inception 模块 |
| ResNet | 2015 | 残差连接，解决深层网络退化 |

## 3. 循环神经网络（RNN）

### 3.1 基本 RNN

适用于序列数据（文本、时间序列等）：

**hₜ = f(Wₕₕhₜ₋₁ + Wₓₕxₜ + b)**

问题：长期依赖和梯度消失/爆炸。

### 3.2 LSTM（长短期记忆网络）

通过门控机制解决长期依赖问题：

- **遗忘门**：决定丢弃哪些信息
- **输入门**：决定存储哪些新信息
- **输出门**：决定输出哪些信息

### 3.3 GRU（门控循环单元）

LSTM 的简化版本，参数更少，训练更快。

## 4. Transformer 架构

### 4.1 自注意力机制（Self-Attention）

**Attention(Q, K, V) = softmax(QKᵀ/√dₖ)V**

- Q（查询）、K（键）、V（值）
- dₖ 是键的维度

### 4.2 多头注意力

将注意力分成多个头，每个头关注不同的特征子空间。

### 4.3 位置编码

由于 Transformer 没有递归结构，需要添加位置信息。

### 4.4 影响

Transformer 彻底改变了 NLP 领域，催生了 BERT、GPT 等模型，并扩展到 CV（Vision Transformer）等领域。

## 5. 生成对抗网络（GAN）

由生成器 G 和判别器 D 组成：

- G 生成假数据
- D 判断数据真假
- 两者对抗训练

应用：图像生成、风格迁移、超分辨率等。

## 6. 关键要点

1. CNN 擅长处理图像数据，通过卷积提取空间特征
2. RNN/LSTM 擅长处理序列数据，通过循环结构捕捉时序关系
3. Transformer 通过自注意力机制并行处理序列，已成为主流架构
4. 深度学习的成功依赖于大数据和强计算力（GPU/TPU）
5. 迁移学习可以在小数据集上取得好效果
